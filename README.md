
🌟 Netflix Homepage Clone

📌 About the Project

This project was created as part of my Frontend Development Internship.

It is a responsive Netflix Homepage Clone built using HTML, CSS, and JavaScript. The project recreates the look and feel of the Netflix landing page with a modern user interface, including a navigation bar, hero banner, movie section, and responsive design.
